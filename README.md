# toronetdeploy

Deploy smart contracts to ToroNet.

## Installation

Global:

```
npm install -g toronetdeploy
```

Or install locally and run via `node index.js`.

## Usage

Run the CLI (if installed globally):

```
toronetdeploy
```

Or run directly:

```
node index.js
```
